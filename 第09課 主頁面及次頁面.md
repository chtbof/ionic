# 第09課 主頁面及次頁面


#### 使用blank樣板(如必要, 請參考第02課):
```
ionic start myApp blank --v2
```


#### 追加一個頁面, 名稱為 detail:
```
cd myApp
ionic g page detail
```


#### 本測驗使用內容如下:
```
 d:\
  |___ <myApp> 
          |___ config.xml 
          |
          |___ <resources>     
          |      |___ <android>  
          |               |___ <icon>
          |               |      |___ 多個.png圖檔          
          |               | 
          |               |___ <splash> 
          |                      |___ 多個.png圖檔              
          |               
          |___ <src>
                 |___ <app> 
                 |       |___ app.module.ts
                 |
                 |___ <assets> 
                 |       |___ <images> 
                 |               |___ a.jpg (300*300px) 
                 |               |___ b.jpg (300*300px) 
                 |               |___ c.jpg (300*300px) 
                 |                 
                 |___ <pages>   
                 |       |___ <home> 
                 |       |      |___ home.html 
                 |       |      |___ home.ts                          
                 |       |      |___ home.scss 
                 |       |
                 |       |___ <detail> 
                 |              |___ detail.html 
                 |              |___ detail.ts                             
                 |
                 |___ <theme>   
                         |___ variables.scss
```
