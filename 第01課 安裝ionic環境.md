# 第01課 安裝ionic環境


#### ionic是一個用來建構App的開放框架, 可以用來產生iOS及Android App. 
這裡的測試使用 Windows 7 作業系統, 主要生成Android App. 


## (1)下載及安裝 Java

Java Development Kits(JDK, Java開發套件) 可以在 `http://www.oracle.com/technetwork/java/javase/downloads/index.html` 下載. <p>
圖示如下:<p>
![GitHub Logo](/images/f01-01.jpg)

安裝完成後增加JAVA_HOME設定: 在[控制台] -> [系統及安全] -> [系統] 畫面中, 點選[進階系統設定]. 
在其畫面中的[進階] -> [環境變數] 畫面中, 新增一個系統變數, 變數名稱為 JAVA_HOME, 變數值為 JDK安裝路徑.
圖示如下:<p>
![GitHub Logo](/images/f01-02.jpg)
