# 第05課 顏色配置


#### 使用blank樣板(如必要, 請參考第02課):
```
npm start myApp blank --v2
```


#### 本測驗使用內容如下:
```
 d:\

          |___ config.xml 
          |
          |___ <resources>     
          |      |___ <android>  
          |               |___ <icon>
          |               |      |___ 多個.png圖檔          
          |               | 
          |               |___ <splash> 
          |                      |___ 多個.png圖檔              
          |               
          |___ <src>          
                 |___ <pages>   
                 |       |___ <home> 
                 |              |___ home.html 
                 |              |___ home.scss   
                 |
                 |___ <theme>   
                         |___ variables.scss   
```
