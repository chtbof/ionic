# 第03課 修改圖示及起始畫面.md



#### 使用blank樣板(如必要, 請參考第02課):
```
ionic start myApp blank --v2
```

#### 本課程使用內容如下:
```
 d:\
  |___ <myApp> 
          |___ <resources>
                    |___ <android>     
                            |___ <icon> 
                            |       |__ (內存圖示檔)
                            |        
                            |___ <splash>  
                                    |__ (內存起始畫面檔)                            
```


#### 修改<icon>及<splash>檔案內容, 再重建apk即可.

```
cordova platform add android 
cordova build
```

#### 測試圖檔可下載, 檔名: [下載/ionic-測試-03.zip].
